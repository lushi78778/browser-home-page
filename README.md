# BrowserHomePage

## 简介

一个定制化的简洁轻巧的起始页。

演示地址：https://lushi78778.github.io/browser-home-page/

## 开始使用

- 直接部署在nginx下

- npm

```bash
# 安装依赖
npm install

# 运行
npm start
```

### 最佳部署

- fork仓库
- 修改后推送，会触发自动部署GitHub pages

PS:配置文件：.github/workflows/static.yml
